# Project II - Numbers are Scary, but I am Proud
My pride & joy! An interactive website that features data given by the Animation Guild on job disruption due to the rise of AI.
It consists of an interactive USA map where you can hover over states for more information & toggle between data sets for different information.

Image

I struggled a lot with this one. I think I was overly ambitious & didn't end up with the result in my head. I learned a lot from this project with all the failing.

HOWEVER! I am ever still so proud of what I was able to make. I did get lots of help from my partner who's a programmer himself. We had lots of fun as I was able to understand more & follow what he speaks about for programming than before! I did also reuse techniques from morse code converter & the weather data visualization by playing around with API-- but wasn't able to successfully apply it all. Turned it around using a combination of Leaflet tutorials & help from my partner for setting up the interactive part. ChatGPT was used to adjust CSS colours, with prompt -- "Modern boxes with each state having it's own unique colour when different dataset selected."

OG Code: Leaflet Open Source Javascript Library.
OG code modified by me with assistence from Tony Mallios.
CSS modified by me with assistence from ChatGPT.
